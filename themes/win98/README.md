# A classic win98 style hexo theme.

![screenshot](./screenshot.png)

## Web font

The theme uses [Ark Pixel Font](https://github.com/TakWolf/ark-pixel-font),
12px proportional Simplified Chinese, under the SIL Open Font License 1.1.
During `hexo generate`, the site creates a shared subset, an exact subset for
each page, and small on-demand ranges for dynamic text. Generated files are
WOFF2 and content-hashed so browsers only fetch the glyphs they need.
