第一次实习，正好公寓里有灶台，给自己一个学习做饭的机会。

# 可乐鸡翅

| 步骤 | 做法                                                         |
| ---- | ------------------------------------------------------------ |
| 1    | 鸡翅中清洗干净，然后用厨房用纸吸一下水。                     |
| 2    | 将鸡翅放入一个碗中，加入料酒、生姜片、少许盐和白胡椒粉，拌匀，腌制 15 分钟。 |
| 3    | 热锅凉油，放入鸡翅，用中小火煎至两面金黄。                   |
| 4    | 加入酱油、老抽、可乐，稍微搅拌均匀。                         |
| 5    | 转小火，盖上锅盖，炖煮 20-30 分钟，期间可以翻动一次，确保鸡翅均匀入味。 |

# 油炒鸡胸肉丝

| 步骤 | 做法                                                   |
| ---- | ------------------------------------------------------ |
| 1    | 先把鸡大胸放姜片焯水10分钟，捞出来之后用厨房用纸吸干。 |
| 2    | 手撕成肉丝。                                           |
| 3    | 热锅凉油，油一定要多加，将肉丝放入锅内翻炒。           |
| 4    | 翻炒至鸡大胸肉丝焦黄即可。                             |
